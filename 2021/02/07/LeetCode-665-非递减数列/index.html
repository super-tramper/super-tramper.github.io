LeetCode-665. 非递减数列

给你一个长度为 n 的整数数组，请你判断在 最多 改变 1 个元素的情况下，该数组能否变成一个非递减数列。

我们是这样定义一个非递减数列的： 对于数组中所有的 i (0 <= i <= n-2)，总满足 nums[i] <= nums[i + 1]。

思路

本题一般的思路就是求数列的逆序数。任选两个数字，排在数列中靠前位置的比后面的数字大，称为一个逆序。数组中这样的数字组合的个数即为数列的逆序数。求逆序数的时间复杂度为n^2。

但此题中要求最多改变一个元素，即逆序数超过1时即可返回false，采用以上方法最大时间复杂度为O(n^2)。

试着思考，在碰到相邻数字为逆序时，有两种方法解决：1.将nums[i-1]的值设为nums[i]；2.将nums[i]的值设为nums[i-1]。

无论采用哪种解决方法，若因此产生新的逆序，则数组中肯定存在其它逆序组合，可直接返回false。为避免回溯，采用方法2更好，一次数列遍历即可确定最终结果。
