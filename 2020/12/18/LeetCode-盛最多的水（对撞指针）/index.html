LeetCode-盛最多的水（对撞指针）

给你 n 个非负整数 a1，a2，…，an，每个数代表坐标中的一个点 (i, ai) 。在坐标内画 n 条垂直线，垂直线 i 的两个端点分别为 (i, ai) 和 (i, 0) 。找出其中的两条线，使得它们与 x 轴共同构成的容器可以容纳最多的水。

说明：你不能倾斜容器。

  • n = height.length

思路：用两个指针分别指向数组的头和尾，计算指针所指元素围成区域的面积，比较两个元素的值，将较小值的指针向内移一位，重新计算面积，依次往复直到指针重合结束。在此过程中不断更新最大值。至于为什么这样最大值不会被遗漏，这里讲解了详细原因。
